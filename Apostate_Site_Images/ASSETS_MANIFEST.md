# THE APOSTATE — IMAGE ASSETS MANIFEST
### Drop the `images/` folder into the site's `/public/assets/` (or `/assets/`). Read the rules first.

This site was built deliberately austere — typographic, no imagery. Images enter in that same register: **few, captioned, restrained, monochrome-treated.** They support the existing copy; they do not redesign the site. The film's thesis is that it names Jason, not Barry — so Jason anchors the imagery and the case material is present but never lurid.

---

## THE RULES

1. **Treat every image to match the austere palette.** Apply a subtle desaturation/duotone toward the site's bone (#F0EDE6) and rust (#B85A38) so images read as one designed family, not a scrapbook. Muted-tone caption (#6B6560) under each, in Inter, small.
2. **`RIGHTS_VERIFY` in a filename = source not yet cleared.** These are news/video-sourced. Render in the local review build, but each emits a row in `/public/assets/LICENSING_CHECKLIST.md` and must be cleared before the site goes to Sam or any external party.
3. **No burned-in news graphics.** The three branded web images (People/Denver7/Sun chyrons) were deliberately excluded. The facts they carried are rebuilt clean — see the MAP instruction in the upgrade prompt.
4. **`excluded/` is not deployed.** The jail image stays out — it's the lurid register the whole project rejects.
